WetServ, a HTTP1.1 server

This website is hosted on a custom made server done by three 42 Lausanne students.
You can find at the bottom some tests on the different methods that the server can work with.

The server is written according to the CPP98 restrictions.
The principle functions are from C but we did use them in the most C++ way.
We used select as main function to manage all the communication and enabling the communications.
All the front-end of the website had been developped as a way to put the server under some real tests and
we are aware that the "quality" of it is questionable.
The goal of the project is the server, everything else is just something we wanted to implement to have a more visual way of testing.

If you need any documentation regarding the project, you can find it here.
